这是“真正给顾客看的版本”。

它和之前的区别：
1. 顾客页面不允许自己注册
2. 顾客页面不允许自己填额度
3. 顾客只能输入你给他的“访问码”来使用
4. 你自己单独用管理员页面开顾客账号和充值

====================
一、你自己怎么用
====================
1. 像之前一样改好 .env
2. 启动 start_windows.bat
3. 顾客页面地址： http://127.0.0.1:8000
4. 管理员页面地址： http://127.0.0.1:8000/admin

====================
二、第一次必须做的事
====================
打开 .env ，把这一行改掉：
ADMIN_PASSWORD=123456

比如改成：
ADMIN_PASSWORD=888999

这个密码只有你自己知道。

====================
三、你怎么给顾客开账号
====================
1. 打开管理员页面 /admin
2. 输入管理员密码
3. 输入顾客名字
4. 输入给他多少点
5. 点“创建顾客访问码”
6. 系统会给你一个访问码
7. 你把访问码发给顾客

====================
四、顾客怎么用
====================
1. 顾客打开首页 /
2. 输入你发给他的访问码
3. 查看剩余额度
4. 贴代码 / 贴报错
5. 点击发送问题

====================
五、顾客用完了怎么办
====================
你打开 /admin
找到他的访问码
给他充值就行。
